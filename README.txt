
--------------------------------------------------------------------------------
							Program Name
--------------------------------------------------------------------------------

yosh:

	This program is a simple unix shell.

Coder: Jun Mun

--------------------------------------------------------------------------------
							Compilation/Run
--------------------------------------------------------------------------------

compile using Makefile:
    make



to run:
   yosh

--------------------------------------------------------------------------------
							Builtin commands
--------------------------------------------------------------------------------

cd [dir]: Changes the working directory to dir.


exit: Exits the shell if there are no running background processes.

help: lists all the built in commands to standard output.

history: lists the last 10 commands given to yosh.

jobs: lists all running background processes.

kill %[num]: Terminates the process corresponding to num(in jobs list)
